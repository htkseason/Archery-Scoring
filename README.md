# Archery Target Scoring  
Use multi-camera views to score an archery target automatically.  
  
---  
  
### Instructions  
The project relies on [OpenCV 3.2.0](http://opencv.org/releases.html) and my [VisionModelLib](https://github.com/htkseason/VisionModelLib).
